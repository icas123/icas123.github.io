# 🎨 Isaac — Página Personal

> Página web personal que presenta mi vida, gustos y pasiones: arte, danza, música y viajes.

---

## 📸 Vista previa

La página está construida como un único archivo HTML autocontenido con todas las imágenes embebidas en Base64, sin necesidad de servidor ni dependencias externas.

---

## 🧭 Secciones

| Sección | Descripción |
|---|---|
| *Hero* | Presentación principal con nombre, ciudad y pasiones |
| *Sobre mí* | Mi historia: ballet, Barranquilla, arte y viajes |
| *Mis Pasiones* | Tarjetas de arte, danza, viajes, música, patinaje y dibujo técnico |
| *Galería de pinturas* | Mis 4 cuadros originales pintados a mano |
| *Mis Viajes* | Fotos reales de París, Florencia, Venecia, Pisa y Milán |
| *Música* | Instrumentos actuales y futuros |

---

## 🛠️ Tecnologías

- *HTML5* — estructura semántica
- *CSS3* — animaciones, variables CSS, grid layout, efectos hover
- *JavaScript vanilla* — Intersection Observer para animaciones al hacer scroll
- *Google Fonts* — Playfair Display + Josefin Sans
- *Imágenes en Base64* — sin dependencias externas, todo en un solo archivo

---

## 🚀 Cómo usar

No requiere instalación ni servidor. Solo abre el archivo en tu navegador:

bash
# Clona el repositorio
git clone https://github.com/tu-usuario/tu-repo.git

# Abre el archivo
open isaac.html


O simplemente arrastra el archivo isaac.html a tu navegador.

## 🎨 Sobre el diseño

El diseño sigue una estética *oscura y dorada* inspirada en galerías de arte y publicaciones editoriales de lujo:

- Paleta de colores: crema cálido, dorado #c9a84c, óxido y verde oscuro
- Tipografía: Playfair Display (serif elegante) + Josefin Sans (sans limpio)
- Animaciones: fade-in al cargar y reveal progresivo al hacer scroll
- Layout: CSS Grid asimétrico, sin frameworks externos
